# Подсказка по Git

Создание репозитория:
```sh
git init
```

Добавление файла:
```sh
git add
```

Добавление коментария:
```sh
git commit -m "Сообщение"
```
Показывает текущее состояние репозитория.
```sh
git status
```

Показывает журнал коммитов
```sh
git log
```
Показывает журнал коммитов строкой
```sh
git log --onelinie
